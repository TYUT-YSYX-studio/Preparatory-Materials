## 设计宗旨

本前置讲义在“一生一芯”计划现有教学内容的基础上，结合太原理工大学的课程培养方案，专为大一零基础学生设计了一套**前置学习路径**，将预学习阶段进一步分为【适应期学员、见习学员、正式学员】三个阶段，通过明确的阶段目标与正反馈激励机制，帮助学员稳步建立学习信心。并融入部分扩展任务，进一步夯实计算机基础能力，讲义难度循序渐进，以基础学习为主，降低学习门槛。
